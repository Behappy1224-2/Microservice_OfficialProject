1.Set the Environment Variable ($env:):

$env:LINE_CHANNEL_ACCESS_TOKEN = 'your_actual_access_token'

(2).(Optional)Rebuild the Docker Image (Only If Changes Were Made):

docker build -t line-microservice .


3.Run the Docker Container:

docker run -e LINE_CHANNEL_ACCESS_TOKEN=$env:LINE_CHANNEL_ACCESS_TOKEN -p 5000:5000 line-microservice


4.Restart Ngrok:

ngrok http 5000


5.Update the Line Webhook URL (if needed):

Ngrok generates a new URL each time it starts, so you may need to update the Webhook URL in the Line Developer Console if the URL changes.
